# Excel Download

> Module: D - Back-End Development / Difficulty: Difficult

Import the provided `database.sql` to set up the database.

When you access the page, the `posts.xlsx` file will be downloaded.

The Excel file should display the id and title of all posts and the number of related comments. (Refer to the format below)

| id | title        | comment count |
|----|--------------|---------------|
| 1  | post title 1 | 3             |
| 2  | post title 2 | 1             |

> Marking aspect:
 - The file posts.xlsx is downloaded. 0.30
 - The downloaded file can be opened with an Excel program. 0.20
 - The data stored in the file reflects the data from the database. 0.50
