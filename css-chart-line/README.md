# CSS Line Chart

> Module: B - Layout / Difficulty: Normal

Implement a line chart using only HTML and CSS.

At first, all values are 0 (the lines and points are at the bottom), but when you hover over the chart area, the lines and points rise from the bottom.

The four items have values from left to right: 65%, 25%, 85%, 40%.

(You can use html and css, but not javascript)

(Refer to resource/demo.mp4)

---

> Marking aspect:
 - There is a chart area, and lines and points are implemented within it. 0.10
 - Initially, the lines and dots are located at the bottom. 0.20
 - When you hover your mouse over the chart area, lines and strokes will rise from the bottom as shown in the provided video. 0.40
 - When you hover your mouse over the chart area, All items will rise to the specified height. 0.10
 - The point is located exactly where the line bends. 0.20
