# Pointer tail

> Module: C - Front-End Development / Difficulty: Normal

Create a new pointer to replace the existing mouse cursor.

The new pointer is a red circle, and when it moves, it has a tail-like line that follows it.

As the mouse moves and the line appears, the pointer gradually gets smaller, and when the line disappears, it returns to its original size.

(Refer to demo.mp4)

> Marking aspect:

 - The default mouse cursor is hidden and replaced with a new red cursor. 0.20
 - The cursor's circle becomes smaller when the mouse is moved. 0.20
 - A tail follows the previous path of the cursor when the mouse is moved. 0.30
 - The faster the mouse is moved, the longer the tail becomes. 0.30
