# Scroll Animation

> Module: B - Layout / Difficulty: Normal

Write animation.css so that the following animations work simultaneously with page scrolling. (Refer to demo.mp4)

You cannot modify any files other than animation.css.

### Section 1
The background narrows and the text disappears.

### Section 2
When the Section 2 area fully fills the screen, it scrolls horizontally to the right.

### Section 3
After all content in Section 2 is scrolled, it scrolls normally, revealing the text.
