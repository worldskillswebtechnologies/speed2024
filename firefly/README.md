# Firefly

> Module: C - Front-End Development / Difficulty: Normal

There are 125 cubes arranged in a 5x5x5 3D grid.

Each cube lights up in sequence: first the center cube, then the surrounding 26 cubes, and finally the outermost 98 cubes.

You can rotate a set of cubes by dragging with the mouse to view them from different angles, and you can zoom in and out using the scroll wheel.

(Refer to demo.mp4)

> Marking aspect:
 - 125 cubes are arranged in a cubic form within a 3D grid. 0.40
 - The cubes light up sequentially from the innermost cube to the outermost cube. 0.20
 - You can rotate the cube set to view it from different angles by dragging the mouse. 0.20
 - You can zoom in and out towards the center using the scroll wheel. 0.20
