# Upload Progressbar

> Module: C - Front-End Development / Difficulty: Easy

After selecting a file, pressing the upload button sends a request to upload.php with that file, displaying the upload progress via a progress bar.

The upload progress should be affected by the network conditions.

> Marking aspect:
 - After selecting a file, clicking the upload button sends the file to upload.php. 0.20
 - When the upload is complete, the progress bar is filled. 0.10
 - The upload progress is displayed as a progress bar, reflecting the actual network conditions. 0.70

> To SCM

During evaluation, browser throttling can be used to verify if the progress is actually affected by the network conditions.
