# Excel Table

> Module: C - Front-End Development / Difficulty: Easy

An Excel table is displayed where you can enter data into cells.

You can see 600 cells at once, with 20 columns and 30 rows.

You can view all cells through scrolling and enter data into the cells.

Columns are labeled with alphabets from A to XFD.

Rows are labeled with numbers up to 1,048,576.

All operations, including data entry and scrolling, should work immediately without delay.

> Marking aspect:
 - View and enter data for 600 cells arranged in 20 columns and 30 rows on one screen. 0.10
 - Scroll to check values ​​up to column XFD and row 1,048,576. 0.10
 - Data in entered cells remains visible even after scrolling off the screen. 0.20
 - All operations, including data entry and scrolling, are executed immediately without delay (no pause). 0.60
