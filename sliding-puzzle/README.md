# Sliding Puzzle

> Module: C - Front-End Development / Difficulty: Easy

Create a 4x4 sliding puzzle using the provided `image.jpg`.

The initial state should have one empty space at the bottom right, and you can click on the pieces around the empty space to move them into the empty space.

When the pieces move, they should move with a smooth animation.

> Marking aspect:
 - In the initial state, a 4x4 puzzle using the given image is displayed (the bottom right piece is missing). 0.30
 - When you click on a piece adjacent to the empty spot, the clicked piece moves to the empty spot. 0.30
 - The movement occurs with a sliding animation. 0.20
 - If there is no empty spot adjacent to the clicked piece, nothing happens. 0.20
