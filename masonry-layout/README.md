# Masonry Layout

> Module: C - Front-End Development / Difficulty: Normal

Implement a gallery layout with 4 columns.

Images are filled into the columns that are the least filled, in order from the leftmost column.

There are 100 images, and 10 images are loaded at a time (in order from 1).

When the last image is reached, another 10 images are loaded. (If the last image is visible even after loading, it will load again)

(Refer to demo.mp4)

> Marking aspect:
 - A gallery layout with 4 columns is displayed. 0.20
 - 100 images are arranged in order (images are placed in the column with the smallest height first, and if columns have the same height, they are filled from the leftmost column). 0.30
 - When the last image is visible, the next 10 images are loaded. 0.30
 - If the last image is still visible after 10 images are loaded, the next 10 images are loaded immediately. 0.20
