# Digital Display Board

> Module: C - Front-End Development / Difficulty: Normal

There is a form to enter text and a place where the entered text will slide in a dot format.

The entered text slides from right to left, and when it completely slides out of view, it reappears on the right and slides again.

The text appears in a perfect dot shape, and the shape of the entered characters is displayed in a single color. (See the example below)

> Marking aspect:

 - There is a form to input text and a slide area. 0.10
 - When you enter text, the input text is displayed in dot form in the slide area (including emojis). 0.50
 - The dots are a single color and have a perfect dot shape. 0.10
 - The dot-form text slides from right to left, and when it completely disappears from view, it slides again from right to left (infinite loop). 0.30
