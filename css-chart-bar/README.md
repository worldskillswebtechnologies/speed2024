# CSS Bar Chart

> Module: B - Layout / Difficulty: Easy

Implement a bar chart using only HTML and CSS.

At first, the screen will be blank, but when you hover the mouse over the chart area, the bars will rise sequentially from the bottom, starting from the left bar.

The values and colors of each data point are not important.

(You can use html and css, but not javascript)

(Refer to resource/demo.mp4)

---

> Marking aspect:
 - The bar graph area is displayed and the initial screen is blank. 0.30
 - When you place the mouse over the chart area, the bars move up in order from the bottom. 0.40
 - When you place the mouse over the chart area, the bar goes up sequentially starting from the left. 0.30
