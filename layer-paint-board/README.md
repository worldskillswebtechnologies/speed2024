# Layer Paint Board

> Module: C - Front-End Development / Difficulty: Difficult

You need to create a paint board that displays all layers and a layer list where each layer can be viewed.

You should be able to change the order of layers, add, delete, and select layers, and the drawings on each layer should also be displayed.

When you select a layer and a color, and draw on the paint board, the drawing is applied to the selected layer.

Any changes should be immediately reflected in the layer.

---

> Marking aspect:
 - There is a paint board showing all layers, a layer list where you can see each layer, and a layer color picker. 0.15
 - Can change the order of layers, add layers, delete layers, and select layers. 0.15
 - When you select a layer and choose a color to draw on the paint board, the drawing is applied to the selected layer. 0.15
 - The drawing on each layer is displayed. 0.275
 - Any changes are immediately reflected in the layer. 0.275
