# WorldSkills Lyon 2024 - Speed Test Tasks

This repository contains the Speed Test Tasks submitted by South Korea for the **WorldSkills Lyon 2024** competition.

## Project Structure

Each task directory is organized as follows:

- **README.md**: Detailed explanation and requirements for the specific task.
- **resource**: Sample files and starting materials provided for the task.
- **media**: Images, videos, and other media assets required for development.
- **answer**: Reference solutions and completed files for the task.
