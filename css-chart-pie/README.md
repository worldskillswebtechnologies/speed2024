# CSS Pie Chart

> Module: B - Layout / Difficulty: Easy

Implement a pie chart using only HTML and CSS.

At first, it will be a blank screen, but when you hover over the chart area, the pie chart will appear in a clockwise direction, and when the mouse leaves the chart area, the chart will disappear in a counterclockwise direction.

The three items have values of 45%, 25%, 30% clockwise.

(You can use html and css, but not javascript)
